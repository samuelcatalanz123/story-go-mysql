## CV — entrada de proyecto

### Español
**Story Admin — Aplicación web full-stack (Go + React)**
Aplicación de gestión de personajes, lugares y escenas con API REST y panel de
administración.
- Desarrollé una **API REST en Go** (net/http, database/sql) con **MySQL**,
  usando arquitectura en capas (handler → service → repository) e inyección de
  dependencias.
- Construí el **frontend en React 19 + TypeScript** (Vite) con un sistema de
  diseño propio en CSS Modules: tablas, formularios en modal, notificaciones y
  diseño responsive.
- Implementé **pruebas automatizadas** (Vitest + Testing Library y `go test`) y
  un flujo de trabajo con Git por ramas y revisión de código.
- Preparé el **despliegue como servicio único** (Docker multi-etapa, migraciones
  automáticas) listo para producción.

### English
**Story Admin — Full-stack web app (Go + React)**
CRUD application to manage story characters, locations and scenes, with a REST
API and an admin dashboard.
- Built a **REST API in Go** (net/http, database/sql) with **MySQL**, using a
  layered architecture (handler → service → repository) and dependency injection.
- Developed the **React 19 + TypeScript** frontend (Vite) with a custom CSS
  Modules design system: data tables, modal forms, toast notifications and a
  responsive layout.
- Wrote **automated tests** (Vitest + React Testing Library and `go test`) and
  followed a branch-based Git workflow with code review.
- Prepared a **single-service deployment** (multi-stage Docker, automatic DB
  migrations), production-ready.
